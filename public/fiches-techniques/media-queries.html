Media Queries

Définition concise

Les media queries sont une fonctionnalité CSS qui permet d'adapter le style des pages web en fonction des caractéristiques de l'appareil, comme la largeur de l'écran, la résolution ou l'orientation.

Contexte d'utilisation

Utilisez les media queries pour :

  • Créer des sites responsives adaptés à tous les écrans (mobiles, tablettes, desktop)
  • Modifier la disposition, la taille de texte ou des images selon la taille de l'écran
  • Améliorer l'expérience utilisateur sans modifier le HTML

Exemple de code commenté


/* Styles généraux */
.container {
    display: flex;
    flex-direction: row;
    gap: 20px;
}

/* Media Query pour les écrans ≤ 768px */
@media (max-width: 768px) {
    .container {
        flex-direction: column; /* passe les éléments en colonne sur mobile */
        gap: 10px;              /* réduit l'espacement */
    }
}

Cas d'usage réel

Dans mon portfolio, j'ai utilisé les media queries pour :

  • Réduire la taille des images et du texte sur mobile
  • Passer la navigation de sidebar à barre horizontale
  • Repositionner les sections “About me” et “Projects” pour mobile

Pièges à éviter

  • Ne pas surcharger le CSS avec trop de media queries
  • Tester sur différents appareils pour vérifier le rendu
  • Préférer les breakpoints standard (480px, 768px, 1024px) pour plus de cohérence

Analyse personnelle

Les media queries m'ont permis de rendre mon portfolio accessible et lisible sur tous les écrans. Avant de les utiliser, mon site était très bien sur desktop mais illisible sur mobile. Grâce à elles, je peux ajuster la mise en page et le design sans toucher au HTML.
