Projet LearnWords React

Nom du projet

LearnWords – Application de gestion de vocabulaire en React

Contexte

LearnWords est un projet réalisé pour consolider les compétences front-end et back-end : gestion de mots avec traduction et catégorie, ajout, édition et suppression des mots, interface responsive pour desktop et mobile, déploiement fonctionnel sur Netlify.

Technologies et choix techniques

  • React 18 : gestion de l’état avec useState, composants réutilisables (WordCard, WordTable, AddWordForm)
  • JavaScript/JSX : logique métier et manipulation du DOM via React
  • HTML5 / CSS3 / SCSS : layout responsive avec Flexbox et Grid
  • API : fetch pour récupération et mise à jour des mots
  • Netlify : déploiement et hébergement du site
  • Bonnes pratiques : séparation des composants, noms clairs, commentaires, responsive design
  • Git : commits réguliers et messages clairs

Problèmes rencontrés et solutions

  • Conflit de versions React (19 vs 18) avec @testing-library/react : corrigé en passant à React 18.2.0
  • Peer dependencies npm : utilisé npm install --legacy-peer-deps
  • Responsive design : ajustements de Grid et Flexbox pour mobile/tablet
  • Déploiement Netlify : configuration correcte du build command et publish directory

Fonctionnalités

  • Affichage des mots dans une table et sous forme de cartes
  • Ajout, modification et suppression de mots
  • Validation pour les trois champs : Mot, Traduction, Thème
  • Message d’erreur si l’utilisateur tente d’ajouter un mot avec un champ vide : "Tous les champs doivent être remplis"
  • Responsive sur desktop, tablette et mobile
  • Gestion des erreurs lors des appels API

Accessibilité (WCAG)

  • Boutons et formulaires avec aria-label
  • Navigation clavier possible sur les listes de mots
  • Contraste des couleurs suffisant pour la lecture

SEO et performance

  • Titres et meta-tags pour chaque page
  • Images avec attribut alt
  • Minification via react-scripts build

Sécurité

  • Validation des inputs pour éviter les injections
  • HTTPS fourni par Netlify
  • Pas de stockage d’informations sensibles côté client

Déploiement

Hébergement sur Netlify : https://learnwordsreactex.netlify.app

Commandes locales :


git clone https://github.com/Taticornaz/my-cards.git
npm install
npm start  # pour lancer en local
npm run build  # pour production

Guide d’utilisation

  1. Ouvrir l’application en local (npm start) ou en ligne (Netlify)
  2. Ajouter un mot et sa traduction via le formulaire
  3. Cliquer sur une carte pour marquer comme complété
  4. Double-clic pour supprimer un mot
  5. Les mots sont affichés sous forme de liste et cartes
