Flexbox

Flexbox pour la mise en page responsive

Définition concise

Flexbox (Flexible Box Layout) est un module CSS qui permet de disposer facilement les éléments dans un conteneur. Il simplifie l'alignement, la répartition de l'espace et la création de mises en page adaptatives sans utiliser les anciennes techniques comme float ou position absolue.

Contexte d'utilisation

Flexbox est particulièrement utile pour :

  • Créer des barres de navigation horizontales ou verticales
  • Aligner et centrer des éléments horizontalement et verticalement
  • Créer des grilles de projets responsives
  • Modifier l'ordre visuel des éléments sans changer le HTML

Exemple de code commenté


/* Conteneur Flexbox */
.container {
    display: flex;           /* transforme le conteneur en flex */
    justify-content: center; /* alignement horizontal */
    align-items: center;     /* alignement vertical */
    gap: 20px;               /* espace entre les éléments */
}

/* Bloc à l'intérieur du conteneur */
.item {
    flex: 1;                 /* chaque bloc prend un espace égal */
    padding: 20px;
    background-color: #5f86cd;
    color: white;
    border-radius: 8px;
}

Cas d'usage réel

Dans mon portfolio, j'ai utilisé Flexbox pour :

  • La barre de navigation (sidebar) : les liens sont alignés verticalement avec display: flex; flex-direction: column;
  • La grille de projets sur la page d'accueil : chaque carte projet est un div dans un conteneur display: flex; flex-wrap: wrap; gap: 20px;
  • Le bloc “About me” : les blocs de texte et images sont alignés horizontalement et centrés verticalement avec align-items: center;

Pièges à éviter

  • Ne pas confondre justify-content et align-items : j'avais inversé ces propriétés pour la barre “About me”, ce qui décalait le texte.
  • Attention aux conteneurs imbriqués : sur la grille de projets, j’avais oublié flex-wrap: wrap; et les cartes dépassaient du conteneur.
  • Ne pas surcharger un conteneur avec trop de propriétés flex ; garder simple pour faciliter la maintenance.

Analyse personnelle

L'utilisation de Flexbox a beaucoup simplifié la mise en page de mon portfolio. Avant, j'utilisais parfois position: absolute pour centrer ou aligner certains blocs, ce qui causait souvent des problèmes d'adaptation sur différents écrans. Avec Flexbox, je peux aligner verticalement et horizontalement les sections “About me” et les cartes projets de manière beaucoup plus simple et responsive. J'ai appris qu'il est important de tester chaque conteneur individuellement avant de combiner plusieurs propriétés.
