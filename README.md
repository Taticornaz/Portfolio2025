# Portfolio2025
Un site portfolio personnel présentant mes projets, compétences et réflexions.

![Capture d'écran du Portfolio](/public/assets/images/myportfolioscreenshot.png)

## Description
Ce portfolio contient :
* Les projets que j'ai réalisés (dossier `Projets`)
  - [Projet 1]
  - [Projet 2]
* Des fiches techniques et ressources d'apprentissage (dossier `fiches-techniques`)
  - [Accessibilité WCAG]
  - [API Fetch en JavaScript]
  - [Flexbox CSS]
  - [HTML5 sémantique]
* Des réflexions personnelles et bilans (dossier `reflexions`)
  - [Mi-parcours]
  - [Bilan final]

Ce projet sert de vitrine de mon apprentissage en développement web front-end et de mon organisation de travail.

## Leçons apprises
* Structurer un site web et un portfolio de manière claire.
* Améliorer mes compétences en HTML et CSS.
* Comprendre les bases de JavaScript.
* Utiliser Git et GitHub pour le versionnage de projets.
* Découvrir l'environnement Node.js et npm.

## Tech Stack
**Client :** HTML5, CSS3 (Flexbox, Grid), JavaScript 
**Server :** Node v13, Express

## Installation
Installer le projet localement avec npm :

```bash
git clone https://github.com/tatia/Portfolio2025.git
cd Portfolio202
npm install 
npm run start

``` 
## Build
Pour construire le projet, exécuter :
npm run build
## Tests
Pour lancer les tests :
npm run test
