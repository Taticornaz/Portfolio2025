# Portfolio2025
Un site portfolio personnel présentant mes projets, compétences et réflexions.

![Capture d'écran du Portfolio](/Portfolio2025/public/assets/images/myportfolio.png)

## Description
Ce portfolio contient :
* Les projets que j'ai réalisés (dossier `projets`)
  - [Gym Website](public/projets/gym/index.html)  
    **Objectifs :** Landing page responsive pour un site de fitness  
    **Fonctionnalités :** Navigation, formulaire de contact, animations simples  
    **Technologies :** HTML5, CSS3, JavaScript, Flexbox  

  - [Motivational Quotes](public/projets/citation/index.html)  
    **Objectifs :** Afficher des citations aléatoires  
    **Fonctionnalités :** Génération aléatoire, design responsive  
    **Technologies :** HTML5, CSS3, JavaScript  

  - [Personal Profile](public/projets/profile/index.html)  
    **Objectifs :** Créer une page profil moderne et responsive  
    **Fonctionnalités :** Sections claires, mise en page Tailwind CSS  
    **Technologies :** HTML5, CSS3, Tailwind CSS  

  - [Quiz App](public/projets/quiz/index.html)  
    **Objectifs :** Tester ses connaissances avec des quiz interactifs  
    **Fonctionnalités :** QCM, feedback immédiat, score final  
    **Technologies :** HTML5, CSS3, JavaScript  

  - [Weather App](public/projets/weather/index.html)  
    **Objectifs :** Afficher la météo en temps réel  
    **Fonctionnalités :** Recherche par ville, icônes météo, températures  
    **Technologies :** HTML5, CSS3, JavaScript, API météo  

  - [Learn Words App (React)](https://learnwordsreactex.netlify.app/)  
    **Objectifs :** Créer une application de flashcards pour apprendre du vocabulaire  
    **Fonctionnalités :** Composants React, gestion de l’état, cartes interactives  
    **Technologies :** React, JavaScript, HTML5, CSS3  

* Des fiches techniques et ressources d'apprentissage (dossier `fiches-techniques`)
  - Accessibilité WCAG
  - API Fetch en JavaScript
  - Flexbox CSS
  - HTML5 sémantique
* Des réflexions personnelles et bilans (dossier `reflexions`)
  - Mi-parcours
  - Bilan final

Ce projet sert de vitrine de mon apprentissage en développement web front-end et de mon organisation de travail.

## Leçons apprises
* Structurer un site web et un portfolio de manière claire.
* Améliorer mes compétences en HTML et CSS.
* Comprendre les bases de JavaScript et React.
* Utiliser Git et GitHub pour le versionnage de projets.
* Découvrir l'environnement Node.js et npm, et résoudre les conflits de dépendances (versions différentes de React).

## Tech Stack
**Client :** HTML5, CSS3 (Flexbox, Grid), JavaScript,React
**Server :** Node v13, Express

## Installation
Installer le projet localement avec npm :

```bash
git clone https://github.com/tatia/Portfolio2025.git
cd Portfolio2025
npm install 
npm run start

``` 
## Build
Pour construire le projet, exécuter :
npm run build
## Tests
Pour lancer les tests :
npm run test
