Animation de texte dynamique

Définition concise

Cet effet utilise JavaScript pour afficher des textes qui se tapent automatiquement et se suppriment pour afficher le texte suivant. Il permet de dynamiser la page d'accueil en affichant plusieurs rôles ou titres de manière animée.

Contexte d'utilisation

L'effet de texte animé est utile pour :

  • Présenter différents rôles ou compétences de manière interactive
  • Attirer l'attention sur la page d'accueil
  • Créer un effet moderne sans frameworks externes

Exemple de code commenté


// Liste des textes à afficher
const texts = ["Frontend Developer", "Web Designer", "Creative Coder"];
let count = 0;       // index du texte courant
let index = 0;       // index de la lettre courante
let currentText = "";
let deleting = false; // état : suppression ou saisie

function type() {
    currentText = texts[count];

    if (!deleting) {
        // Ajouter une lettre
        document.querySelector(".typing-text").textContent = currentText.slice(0, index + 1);
        index++;

        // Si le mot est complet, attendre puis supprimer
        if (index === currentText.length) {
            deleting = true;
            setTimeout(type, 1500);
            return;
        }
    } else {
        // Supprimer une lettre
        document.querySelector(".typing-text").textContent = currentText.slice(0, index - 1);
        index--;

        // Si tout est supprimé, passer au texte suivant
        if (index === 0) {
            deleting = false;
            count = (count + 1) % texts.length;
        }
    }

    // Vitesse de frappe et suppression
    setTimeout(type, deleting ? 50 : 120);
}

type();

Cas d'usage réel

Dans mon portfolio, j'ai utilisé cet effet pour :

  • Afficher "Frontend Developer", "Web Designer" et "Creative Coder" sur la page d'accueil
  • Dynamiser le header et attirer l'attention de l'utilisateur

Pièges à éviter

  • Bien gérer les délais entre la saisie et la suppression
  • Veiller à ce que la classe ciblée dans le HTML corresponde à celle du script
  • Éviter de surcharger la page avec trop d'effets similaires

Analyse personnelle

Cet effet a rendu ma page d'accueil plus vivante et professionnelle. J'ai appris à manipuler les chaînes de caractères et le DOM de manière simple et efficace. Il est facile à personnaliser et améliore l'expérience utilisateur.J'apprécie cette technique car elle rend le site plus vivant et met en valeur mes différentes compétences. Elle permet également d'attirer l'attention de l'utilisateur sans utiliser trop de JavaScript complexe.
